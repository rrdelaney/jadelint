# Installation

```
npm install -g jadelint
```

# Usage

```
jadelint myFile.jade otherFile.jade myDirectory
```

# Configuration

Put custom rule levels in `.jadelintrc`. Example
```json
{
    "HTMLRootRequiresLang": "ignore"
}
```

# Developing

```
git clone https://github.com/rrdelaney/jadelint
cd jadelint
grunt
```
