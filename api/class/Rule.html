Constructor Details

# (void) constructor(filename, node, conf = {})

Constructor for Rules, sets filename and linted node

To apply configuration to the rule, use the conf argument. If a boolean is provided, the level will be set to 'ignore' if false and 'error' if true. If a string is given, the level is set to that string. If an object is given, any property of the object will be applied to the rule.

Parameters:

  • filename ( String ) — the name of the file being linted
  • node ( Node ) — the jade node being linted
  • conf ( Boolean|String|Object ) — the configuration object to apply to the rule
